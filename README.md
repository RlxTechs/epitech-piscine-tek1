# EPITECH Piscine TEK1 - Training Lab

Projet personnel pour apprendre progressivement les bases de la piscine Epitech / TEK1.

Important : les exercices sont originaux et pedagogiques. Le but est de progresser petit a petit sans copier les sujets officiels.

## Objectifs

- Terminal
- Git et GitHub
- Langage C
- Compilation
- Fonctions
- Pointeurs
- Strings
- Tableaux
- Makefile
- Libmy
- Algorithmes
- Mini-projets

## Progression

00_start_here : premier programme C
01_shell_basics : commandes terminal
02_c_basics : variables, conditions, boucles
03_functions : fonctions simples
04_pointers : pointeurs
05_strings : chaines de caracteres
06_arrays : tableaux
07_libmy : librairie personnelle
08_makefiles : compilation automatisee
09_algorithms : logique et algorithmes
10_mini_projects : vrais petits projets
