# Module 04 - Terminal Game Engine

## Objectif

Créer un mini moteur de jeu terminal en C.

## Compiler

make

## Lancer le jeu interactif

.\terminal_game.exe

## Lancer la démo automatique

.\terminal_game.exe --script

## Lancer les tests

.\unit_tests.exe

## Tout vérifier

powershell -ExecutionPolicy Bypass -File .\check.ps1
