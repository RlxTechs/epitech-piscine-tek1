# Module 06 - Snake Terminal

## Objectif

Créer un mini Snake jouable dans le terminal.

## Compiler

make

## Lancer le jeu

.\snake_game.exe

## Lancer la demo automatique

.\snake_game.exe --demo

## Lancer les tests

.\unit_tests.exe

## Tout vérifier

powershell -ExecutionPolicy Bypass -File .\check.ps1
