# Module 02 - Debug Memory

## Objectif

Apprendre a securiser la gestion memoire en C.

## Structure

include/ : header principal
src/ : fonctions memoire, strings, arrays, debug
tests/ : tests unitaires
main.c : demonstration
Makefile : compilation
check.ps1 : verification automatique

## Compiler

make

## Lancer la demo

.\memory_lab.exe

## Lancer les tests

.\unit_tests.exe

## Tout verifier

powershell -ExecutionPolicy Bypass -File .\check.ps1
