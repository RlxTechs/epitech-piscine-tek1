# Module 03 - Math for Games

## Objectif

Construire une petite base mathematique utile pour les jeux 2D.

## Ce module contient

- clamp
- lerp
- deg_to_rad
- rad_to_deg
- vector2
- distance
- normalize
- dot product
- collisions rectangles
- point dans rectangle
- random int
- random double

## Compiler

make

## Lancer la demo

.\math_game_demo.exe

## Lancer les tests

.\unit_tests.exe

## Tout verifier

powershell -ExecutionPolicy Bypass -File .\check.ps1
