# Module 09 - Game Project Runner

## Objectif

Créer un launcher terminal pour lancer les modules jeux depuis un seul endroit.

## Compiler

make

## Lister les modules

.\game_runner.exe --list

## Lancer un module

.\game_runner.exe --run 1

## Lancer toutes les demos

.\game_runner.exe --demo

## Menu interactif

.\game_runner.exe

## Tests

.\unit_tests.exe

## Verification complete

powershell -ExecutionPolicy Bypass -File .\check.ps1
