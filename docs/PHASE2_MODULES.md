# Phase 2 - Modules serieux TEK1

## Objectif

La phase 1 a servi a apprendre les bases :

- C basics
- fonctions
- strings
- pointeurs
- tableaux
- malloc
- fichiers
- mini printf
- mini ls
- mini shell
- mini grep
- tests

La phase 2 sert a passer a une logique plus professionnelle.

## Nouveaux objectifs

- structurer un vrai projet C
- separer headers, sources, tests et executables
- automatiser les tests sans reecrire check_all_days a chaque fois
- apprendre le debug propre
- apprendre la norme de code
- construire des mini-projets complets
- documenter chaque module
