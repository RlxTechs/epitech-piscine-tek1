# Module 12 - Python Pygame Arcade

## Objectif

Créer une base de jeu arcade Python avec une logique propre, testable, et une interface Pygame optionnelle.

## Important

Les tests fonctionnent sans pygame.

Pour lancer la vraie fenêtre graphique, pygame doit être installé :

python -m pip install pygame

## Contenu

- logique de joueur
- ennemis
- pièces
- score
- collisions
- simulation headless
- tests unitaires
- app graphique optionnelle avec pygame

## Commande de test

powershell -ExecutionPolicy Bypass -File .\modules\module12_python_pygame_arcade\check.ps1
