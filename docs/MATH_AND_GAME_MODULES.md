# Modules mathematiques et jeux

## Pourquoi ajouter des maths ?

Les jeux utilisent beaucoup de notions mathematiques simples :

- position
- vitesse
- distance
- direction
- collision
- hasard
- interpolation
- limites
- angles
- grille

Avant de coder des jeux 2D propres, il faut avoir une base mathematique reutilisable.
