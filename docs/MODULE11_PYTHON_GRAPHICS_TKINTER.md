# Module 11 - Python Graphics Tkinter

## Objectif

Ce module introduit les interfaces graphiques avec Python.

On utilise Tkinter parce qu il est souvent deja disponible avec Python.

## Ce module contient

- une fenetre graphique
- un canvas
- un joueur
- une piece a collecter
- un score
- des controles clavier
- une logique de jeu separee de l interface
- des tests unitaires Python
- un script PowerShell de verification

## Pourquoi Python graphique ?

Python permet de creer rapidement des prototypes graphiques.

C est tres utile pour :

- mini-jeux
- interfaces admin
- outils internes
- visualisation
- prototypes avant version web ou desktop

## Commandes

Lancer les tests :

powershell -ExecutionPolicy Bypass -File .\modules\module11_python_graphics_tkinter\check.ps1

Lancer le jeu graphique :

python .\modules\module11_python_graphics_tkinter\app.py
